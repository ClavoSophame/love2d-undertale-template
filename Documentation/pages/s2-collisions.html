碰撞

……你是否已经厌倦了Create Your Frisk的10*10超大的碰撞？
你是否已经厌倦了需要自己手写碰撞代码，最后还很难算对的事情？
SoulEngine提供了手算的碰撞系统，你可以使用它来创建你想要的碰撞效果。

碰撞图形的初始化

首先，你需要初始化碰撞图形。
也就是说你需要传输一些数据，来定义碰撞图形。

我们将使用collisions.FollowShape(sprite)来初始化碰撞图形。 下面是初始化一个碰撞图形的例子： 

                
                    -- 创建贴图
                    local bullet = sprites.CreateSprite("bullet.png", 1)
                    local player = sprites.CreateSprite("bullet.png", 1)

                    -- 初始化碰撞图形
                    local coll1 = collisions.FollowShape(bullet)
                    local coll2 = collisions.FollowShape(player)
那么对于这个初始化的碰撞图形coll，它存储了以下五个变量：
  • coll.x：碰撞图形的X坐标
  • coll.y：碰撞图形的Y坐标
  • coll.w：碰撞图形的宽度
  • coll.h：碰撞图形的高度
  • coll.angle：碰撞图形的角度

如果你的贴图改变了锚点，不是原先默认的(0.5, 0.5)时，该库会通过矩阵的计算矫正真正的碰撞坐标。
因此不必担心碰撞会出现错误，所有碰撞都是精确计算的。

碰撞的检测

该库提供了矩形与点、矩形与矩形、椭圆与点、椭圆与矩形、椭圆与椭圆的碰撞检测。

  • collisions.RectangleWithPoint(rectangle, point)：矩形与点的碰撞。
  • collisions.RectangleWithRectangle(rectangle1, rectangle2)：矩形与矩形的碰撞。
  • collisions.CircleWithPoint(circle, point)：椭圆与点的碰撞。
  • collisions.CircleWithRectangle(circle, rectangle)：椭圆与矩形的碰撞。
  • collisions.CircleWithCircle(circle1, circle2)：椭圆与椭圆的碰撞。

示例： 
                
                    function love.update(dt)
                        -- 检测两个矩形是否碰撞
                        print(collisions.RectangleWithRectangle(coll1, coll2))
                    end

                    -- 在SoulEngine中，我们使用的是矩形和点的碰撞。
                    -- 也就是说，玩家的碰撞变成了1*1 px。
                    function love.update(dt)
                        -- 检测矩形和点的碰撞
                        print(collisions.RectangleWithPoint(coll1, coll2))
                    end

更为复杂的碰撞

如果你需要更精确的碰撞检测（如多边形碰撞、弹跳、摩擦力等），可以使用 LOVE2D 内置的 Box2D 物理引擎。
它比简单的 AABB（轴对齐碰撞）更强大，适用于复杂的物理模拟。

1. 初始化物理世界

首先，你需要创建一个 物理世界，并设置重力（也可以填写0, 0）： 


            local world
    
            function love.load()
                -- 初始化物理世界（重力：Y轴向下 300 单位）
                world = love.physics.newWorld(0, 0, true)
            end
    
            function love.update(dt)
                -- 更新物理世界（建议传入 dt 以确保稳定模拟）
                world:update(dt)
            end

2. 创建物理物体（刚体）

物理物体可以是 动态的（受物理影响）、静态的（固定不动）或 运动学的（可移动但不直接受力影响）。 


            local playerBody, playerShape, playerFixture
    
            function love.load()
                -- 创建一个动态刚体（可移动、受重力影响）
                assert(world, "物理世界未初始化")
                playerBody = love.physics.newBody(world, 400, 300, "dynamic") or error("无法创建刚体")
    
                -- 定义一个矩形碰撞形状（宽50，高50）
                playerShape = love.physics.newRectangleShape(50, 50)
    
                -- 将形状绑定到刚体，并设置物理属性（密度、摩擦力、弹性）
                playerFixture = love.physics.newFixture(playerBody, playerShape, 1)
                playerFixture:setRestitution(0.5)  -- 弹性系数（0=无弹跳，1=完全弹性）
                playerFixture:setFriction(0.3)    -- 摩擦力（0=无摩擦，1=高摩擦）
            end

3. 碰撞检测（回调函数）

你可以监听碰撞事件，例如： 


            function beginContact(fixtureA, fixtureB, contact)
                -- 当两个物体开始碰撞时触发
                print("碰撞发生！")
            end
    
            function endContact(fixtureA, fixtureB, contact)
                -- 当两个物体结束碰撞时触发
                print("碰撞结束！")
            end

            function preSolve(fixtureA, fixtureB, contact)
                -- 碰撞解决前调用，可用于修改碰撞属性
                -- 例如：contact:setEnabled(false) 可以禁用这次碰撞
            end

            function postSolve(fixtureA, fixtureB, contact, normalImpulse, tangentImpulse)
                -- 碰撞解决后调用，可获取碰撞力度数据
            end
    
            function love.load()
                -- 注册碰撞回调
                world:setCallbacks(beginContact, endContact, preSolve, postSolve)
            end

4. 渲染物理物体

物理引擎只计算碰撞，你仍然需要手动绘制物体： 


            function love.draw()
                -- 获取刚体的位置和角度
                local x, y = playerBody:getPosition()
                local angle = playerBody:getAngle()
    
                -- 绘制矩形（旋转角度由物理引擎计算）
                love.graphics.push()
                love.graphics.translate(x, y)
                love.graphics.rotate(angle)
                love.graphics.rectangle("fill", -25, -25, 50, 50)
                love.graphics.pop()
            end

5. 施加力与冲量

你可以让物体移动或跳跃： 


            function love.keypressed(key)
                if key == "space" then
                    -- 施加向上的冲量（模拟跳跃）
                    playerBody:applyLinearImpulse(0, -500)
                elseif key == "right" then
                    -- 施加向右的力
                    -- 力的单位是牛顿/米/秒²，冲量是牛顿·秒
                    -- 在LOVE2D中，这些值需要根据你的游戏尺度调整
                    playerBody:applyForce(200, 0)  -- 约等于20kg物体受到10m/s²的加速度
                end
            end
