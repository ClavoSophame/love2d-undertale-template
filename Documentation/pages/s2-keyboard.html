玩家不能干预的游戏，可是没办法玩的，所以在这时……
获取到玩家键盘和鼠标、亦或者触屏的操作就显得尤为重要。

键盘的输入与输出

基础函数：

  • keyboard.Bind(name, ...) - 绑定名字和按键，后续可通过检测名字获取其下的所有按键。
    • name - 要绑定的名字。
    • ... - 要给这个名字绑的所有按键。
  • keyboard.GetState(key) - 获取按键的状态
    • key - 可以是绑定过的名称，也可以是单个按键。
    以下为其返回值：
    • 0 - 表示没有按下，但也不是松开的那一瞬间。
    • 1 - 表示刚刚按下的一瞬间。
    • 2 - 表示在刚刚按下的一瞬间之后的长按。
    • -1 - 表示按键刚刚松开的一瞬间。
  • keyboard.AllowPlayerInput(bool) - 设置是否允许接受玩家的按键，可搭配后面的AutoPress函数使用。
    • bool - 是否允许接受玩家的按键，默认为true。
  • keyboard.AutoPress(key, duration) - 执行游戏内无玩家干预的自动按键。
    • key - 要进行自动操作的按键。
    • duration - 按键被按下的时长。
  • keyboard.ReturnLetter() - 返回键盘按下的单个字母/字符。
示例 
                
-- 采用 >0 的方式说明该条件只要按键按下就会被触发。
if (keyboard.GetState("left") > 0) then
    bone.x = bone.x - 1
end

local str = ""
-- SCENE.UPDATE
str = str .. keyboard.ReturnLetter()
if (keyboard.GetState("backspace") == 1) then -- 按下退格键后删除末尾的一个字符。
    str = str:sub(1, -2)
end
