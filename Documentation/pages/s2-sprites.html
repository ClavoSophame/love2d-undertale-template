贴图

您是否曾想过：游戏中的图片是怎么生成的？
那么现在，在SoulEngine的文档中，你来对地方了。

贴图的创建

我们只需要使用一个……类似于Create Your Frisk中贴图的创建方法的一个函数。 

            
                -- 创建一个图层为1、名称为bullet.png的贴图
                local bullet = sprites.CreateSprite("bullet.png", 1)
是不是很简单？ 如果你这么做了，你就会看到一个bullet.png图片出现在了屏幕(320, 240)的位置上。
（很明显因为你没给位置，所以这个函数会默认给你一个位置）

接下来，我们直观地感受一下关于这个贴图的图层顺序。
创建一个新的poseur.png，给予它另一个名字，图层比之前的要高一个。 

            
                -- 创建一个图层为2、名称为poseur.png的贴图
                local poseur = sprites.CreateSprite("poseur.png", 2)
于是你就会神奇的发现，原先创建的bullet.png不见了。
当然，是poseur盖住了原先的bullet。
接下来请自己试试重新修改bullet.png的图层。使其高于poseur。

贴图的属性

当然，贴图是不能只是单纯的一个图片。
我们可能需要修改它的一些属性来达成想要的效果。
比如：改变它的角度、改变它的大小、改变它的颜色……

接下来，我们来尝试一下。 

                
                    -- 创建一个图层为1、名称为bullet.png的贴图
                    local bullet = sprites.CreateSprite("bullet.png", 1)

                    -- 设置bullet的旋转角度为45度.
                    bullet.rotation = 45

                    -- 设置bullet的长为原来的2倍、宽为原来的0.5倍.
                    bullet:Scale(2, 0.5)

                    -- 设置bullet的颜色为红色.
                    bullet.color = {1, 0, 0}

                    -- ...

我将列举出贴图所有属性的值和方法。

贴图的属性

sprite.path
类型：字符串(string)
贴图的路径，例如 "bullet.png"。

sprite.realName
类型：字符串(string)
去掉扩展名后的贴图名称。

sprite.image
类型：Image
Love2D 加载的图像对象。

sprite.layer
类型：字符串/数值
当前贴图所在的图层。（数值越大图层越高）

sprite.x, sprite.y
类型：数值(number)
贴图的坐标。

sprite.width, sprite.height
类型：数值(number)
贴图宽度与高度，自动从图像获取。

sprite.xpivot, sprite.ypivot
类型：数值(number)
锚点位置（范围通常为0-1）。

sprite.rotation
类型：数值(number)
贴图的旋转角度（单位：度）。

sprite.velocity
类型：表(table)
包含 x 和 y 速度分量的表。

sprite.xscale, sprite.yscale
类型：数值(number)
贴图的缩放比例。

sprite.xshear, sprite.yshear
类型：数值(number)
贴图的剪切量。

sprite.alpha
类型：数值(0-1)
透明度。

sprite.color
类型：表(table)
颜色表，格式为 {r, g, b, a}。

sprite.visible
类型：布尔(boolean)
是否可见。

sprite.isactive
类型：布尔(boolean)
是否处于激活状态。

动画属性

sprite.animation
类型：表(table)
包含动画帧序列、间隔、模式等。

特殊效果

sprite.shaders / sprite.stencils
类型：表(table)
着色器和模板缓冲相关配置。

sprite.dust
类型：表(table)
用于“尘埃化”特效，包含贴图像素点的行信息。

贴图的函数

  • sprite:MoveTo(x, y)：设置贴图到指定位置。
  • sprite:Move(dx, dy)：相对移动。
  • sprite:SetSpeed(x, y)：设置移动速度。
  • sprite:Scale(x, y)：设置缩放。
  • sprite:Pivot(x, y)：设置旋转锚点。
  • sprite:Shear(x, y)：设置剪切。
  • sprite:Set(path)：更换贴图图像。
  • sprite:GetPosition()：返回当前坐标。
  • sprite:SetAnimation(frames, interval, mode)：设置动画帧、间隔和模式（loop、oneshot、oneshot-empty）。
  • sprite:SetStencils(stencils)：设置模板缓冲（遮罩）。
  • sprite:SetShaders(shaders)：设置着色器。
  • sprite:Dust(sound, destroy)：触发尘化特效并销毁。
  • sprite:Destroy()：彻底销毁贴图。
  • sprite:Remove()：移除贴图（与Destroy功能相同）。

全局函数

  • sprites.CreateSprite(path, layer, filters)：创建一个新的贴图对象。
  • sprites.Update()：更新所有贴图状态。
  • sprites.Draw()：绘制所有贴图。
  • sprites.clear()：清除所有贴图。

转换弹幕

只是有了贴图，那可不够，这些贴图中有一部分是需要转化为弹幕对玩家造成实质性的伤害的。
对于SoulEngine，我们使用sprite.isBullet = true来将其强制转化为弹幕类型。
示例：
            
                local sprite = sprites.CreateSprite("bullet.png", 1)
                sprite.isBullet = true
那么这时候，这个贴图将被强制转化为弹幕类型，执行对玩家造成伤害的逻辑。
如果要给贴图分属性，例如普通攻击、蓝色攻击、橙色攻击等……在SoulEngine中，我们使用sprite['HurtMode'] = "xxx"的格式来进行属性设置。
示例：
            
                local sprite = sprites.CreateSprite("bullet.png", 1)
                sprite.isBullet = true
                sprite['HurtMode'] = "orange"
该弹幕将变为橙色属性的弹幕。

一些示例

让贴图化成尘埃： 
            
                local sprite = sprites.CreateSprite("bullet.png", 1)
                sprite:Dust(true, true)


创建一个有抗锯齿的贴图： 
            
                local sprite = sprites.CreateSprite("bullet.png", 1, {"linear", "linear"})


创建一个带Shader的贴图（这里举渐变色效果为例）： 
            
                -- 创建渐变Shader对象并设置颜色
                local shader = love.graphics.newShader("Scripts/Shaders/gradient")
                shader:send("color_tl", {1, 0, 0, 1})
                shader:send("color_tr", {0, 0, 1, 1})
                shader:send("color_br", {0, 1, 0, 1})
                shader:send("color_bl", {1, 1, 0, 1})

                -- 创建贴图并设置Shader
                local bul = sprites.CreateSprite("poseur.png", 1)
                bul:SetShaders({shader})


创建一个带有多个贴图动画的贴图： 
            
                -- 创建贴图并设置动画
                local bul = sprites.CreateSprite("bullet.png", 1)

                -- 这将每10帧切换一次，且循环播放。
                bul:SetAnimation({"bullet.png", "poseur.png"}, 10, "loop")

                -- 这将每5帧切换一次，且仅播放一次。
                bul:SetAnimation({"bullet.png", "poseur.png"}, 5, "oneshot")

                -- 这将每5帧切换一次，且仅播放一次，当动画播放完毕后，会清空贴图。
                bul:SetAnimation({"bullet.png", "poseur.png"}, 5, "oneshot-empty")
