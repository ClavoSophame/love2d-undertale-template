玩家

最主要因素。

玩家属性的修改

文件路径：
Scripts/Libraries/Game/Encounter.lua
在这里面，是战斗初始化的时候，玩家的所有设置。

可设置属性

  • sprite - 玩家贴图的创建（颜色修改位于scene_battle.lua中）。
  • name - UI显示的名字。
  • lv - 玩家的等级。
  • maxhp - 玩家的最大HP。
  • hp - 玩家的当前HP。
  • soulMode - 玩家的灵魂模式（1~7彩虹色排列）。
  • canMove - 设置玩家是否可以移动。
  • speed - 玩家灵魂的移动速度。
                
Player = {
    shadows = {},
    sprite = sprites.CreateSprite("Soul Library Sprites/spr_default_heart.png", 15),
    name = "Chara",
    lv = 1,
    maxhp = 20,
    hp = 20,
    soulMode = 1,
    canMove = true,
    speed = 2
}

玩家的方法

  • Player.Hurt(damage, time, sound) - 扣除玩家生命值。
    • damage - 玩家所受伤害。
    • time - 受伤时长（无敌时间）。
    • sound - 是否播放受伤音效（false时可以用作murdersans的盾）。
  • Player.Heal(value, sound) - 回复玩家生命值。
    • value - 回复多少生命值。
    • sound - 是否播放音效（因为播放音效有时候特别吵，可以选择关闭）。
  • Player.SetSoul(number, ...) - 设置玩家灵魂。
    • number - 第几个灵魂（默认只支持1~7）。
    • ... - 额外设置，用作表可以设置蓝魂是否为连跳蓝魂等。
                
-- 设置玩家灵魂。
-- 蓝魂连跳橙魂（可以通过按下键立刻减小跳跃）。
Player.SetSoul(6, {orange = true, decay = true})

新建/修改灵魂运动逻辑

有些时候，您的战斗可能需要各种各样的灵魂，所以您需要学会自己修改灵魂逻辑。
找到encounter.Player.Movement(dt)函数，修改其内部逻辑

一些基本的用法：

  • Player:Move(x, y) - 移动玩家。
  • Player:MoveTo(x, y) - 移动玩家到坐标(x, y)的位置。
